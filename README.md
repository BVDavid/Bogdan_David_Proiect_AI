## Descriere
Acest proiect utilizează algoritmi de Machine Learning (Random Forest) pentru a clasifica traficul de rețea în două categorii: "Normal" sau "Atac". Modelul a fost antrenat pe setul de date NSL-KDD.

## Structura Proiectului
- `main.py`: Scriptul principal care încarcă datele, antrenează modelul și afișează rezultatele.
- `KDDTrain+.txt`: Setul de date utilizat.
- `requirements.txt`: Lista bibliotecilor necesare.

## Instrucțiuni de Instalare și Rulare
1. Asigurați-vă că aveți Python 3 instalat.
2. Instalați bibliotecile necesare:
   ```bash
   pip install -r requirements.txt

3. Rulați scriptul principal:
   ```bash
    python main.py
